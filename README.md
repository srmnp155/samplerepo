# Attractive To-Do List

A simple and attractive to-do list web app with flash notifications for adding and deleting tasks.

## Features

- Add new tasks
- Mark tasks as completed by clicking on them
- Delete tasks with a delete button
- Flash messages for user feedback
- Responsive and modern UI

## How to Use

1. Open `index.html` in any modern web browser.
2. Type a task into the input box and press "Add".
3. Click a task to mark it completed/uncompleted.
4. Click the "Delete" button next to a task to remove it.
5. Flash messages will appear for feedback.

## Technologies Used

- HTML5
- CSS3 (with Google Fonts)
- Vanilla JavaScript
